# INDEX - Backups da extensao Chat Facil

Status: indice inicial criado pelo Codex.

## Protocolo

Toda atualizacao futura da extensao Chat Facil ou de aplicacoes/extensoes relacionadas deve gerar backup versionado em:

`backups/extensoes/chat-facil/YYYY-MM-DD_HHhMM-BRT_vNN_resumo-curto/`

## Regras de seguranca

- Nao apagar backups antigos.
- Nao sobrescrever versoes antigas.
- Nao renomear Chat Bridge/Chat Facil sem confirmacao.
- Registrar funcoes atualizadas, data, versao, origem e destino do backup.

## Entradas

Nenhum backup funcional da extensao foi criado nesta etapa. A etapa atual criou apenas a estrutura de ponte e o indice para uso futuro do Antigravity.
