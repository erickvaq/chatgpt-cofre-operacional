# Funcoes esperadas do Chat Facil

Status: lista funcional planejada para implementacao/revisao no Antigravity.

## Fluxos principais

- Capturar mensagens de uma conversa do ChatGPT.
- Selecionar origem e destino entre abas/conversas.
- Preparar texto antes do envio.
- Enviar texto para outra aba/conversa quando autorizado pelo usuario.
- Repetir capturas recentes.
- Auxiliar backups de regras/memorias.
- Manter log local/historico operacional.

## Indicadores visuais

- Amarelo: texto preparado.
- Vermelho: enviado/processando.
- Verde: resposta concluida/pronta.

## Restricoes

- A extensao e opcional e depende do navegador/perfil atual.
- Nao deve assumir acesso a conversas antigas sem autorizacao.
- Nao deve substituir confirmacao visual real de memoria persistente.
- Nao deve apagar ou sobrescrever backups antigos.
