# Regra Chat Facil

Quando uma tarefa envolver copiar, transferir, capturar, comparar ou organizar conteudo entre conversas, abas ou telas do ChatGPT, considerar a extensao local Chat Facil como opcao pratica quando ela estiver disponivel. O Chat Facil e uma ponte rapida dentro do ChatGPT, nao substitui Codex, Antigravity, GitHub, Drive ou modo Agente.

Nao tratar o Chat Facil como obrigatorio, pois o usuario pode estar no celular, outro computador, outro navegador, outro perfil, conversa limpa ou ambiente sem a extensao instalada.

Quando o Chat Facil estiver disponivel, verificar se ele pode simplificar a tarefa antes de pedir copia manual, prints ou repeticao de texto. Para arquivos, GitHub, scripts, commits e automacao tecnica, priorizar Codex; para programacao, mockups, testes e evolucao da extensao, priorizar Antigravity; para login, permissoes e navegacao visual em sites, usar modo Agente quando autorizado.
