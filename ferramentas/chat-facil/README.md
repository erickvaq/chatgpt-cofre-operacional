# Chat Facil

Status: estrutura de handoff criada pelo Codex para execucao no Antigravity.

## Objetivo

Chat Facil e uma extensao local de Chrome/Edge usada para acelerar fluxos dentro do ChatGPT:

- capturar mensagens;
- transferir texto entre abas/conversas;
- repetir capturas;
- auxiliar backups;
- reduzir copiar/colar manual;
- organizar conteudo operacional entre ChatGPT, Codex, Antigravity e GitHub.

## Papel das ferramentas

- Codex: orquestra, prepara prompts, organiza GitHub, registra backups e devolve links.
- Antigravity: programa, cria mockups, testa workspace e evolui a extensao.
- GitHub: cofre/versionamento.
- ChatGPT: revisa pelos links e valida o fluxo.

## Estado atual

O Codex nao possui controle direto do Antigravity nesta sessao. Por isso, esta pasta foi criada como pacote de ponte/handoff, com instrucao pronta para o Antigravity executar a etapa visual e tecnica.
